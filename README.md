# 🧧 福驍 Bingo 遊戲 - 多人同步版本

> 一個支持多人實時同步的網頁賓果遊戲，支持跨手機、平板、電腦等多設備協作

## ✨ 核心特色

- 🎮 **實時多人同步** - 多個設備上的玩家實時看到彼此的進度
- 📱 **跨設備支持** - 手機、平板、電腦無縫協作
- 🔄 **智能同步** - 三層同步機制確保數據一致性
- 🌐 **雲端儲存** - 使用 Firebase 實現全球實時同步
- 👥 **玩家列表** - 實時顯示線上玩家和連線數
- 💾 **自動保存** - 遊戲數據自動本地和雲端保存

## 🚀 快速開始

### 方式 1：本地測試（無需配置）

```bash
1. 雙擊 index.html 打開遊戲
2. 第 1 個分頁輸入「玩家 A」進入遊戲
3. Ctrl+T 開新分頁，輸入相同 URL
4. 輸入「玩家 B」進入遊戲
5. 在玩家 A 分頁點擊「🐎 抽球號」
6. 玩家 B 的棋盤自動同步！✅
```

### 方式 2：跨設備測試（需要 Firebase）

請參閱 **快速開始.md** - 5 分鐘完成設定

## 🎯 三層同步機制

```
層級 1: LocalStorage
  └─ 同一設備不同分頁間同步
       ↓
層級 2: 遊戲會話 ID
  └─ 識別一局遊戲
       ↓
層級 3: Firebase Realtime Database
  └─ 不同設備全球實時同步
```

## 💡 常見問題

### Q: 本地測試能用嗎？
**A**: 可以！不需要任何配置。直接雙擊 index.html，在不同分頁打開即可同步。

### Q: 怎樣讓手機和電腦一起玩？
**A**: 遵循 **快速開始.md** 中的「跨設備測試」部分。

### Q: 支持多少個玩家？
**A**: Firebase 免費方案支持 100 個同時連線。
